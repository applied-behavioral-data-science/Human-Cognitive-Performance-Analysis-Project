# Data

This folder will contain datasets used in the project.

Raw data should remain unmodified. Any cleaned or processed data will be added later with documentation.

## Dataset
- **Source:** Kaggle
- https://www.kaggle.com/datasets/b2a3bfcf6f3a94ee3cf357ccd7212879dd31d46f879ca3866464a0dd8802ca65

- **Description:**  
This dataset contains self-reported lifestyle and behavioral variables (e.g., sleep, stress, screen time, physical activity, caffeine use) alongside a measure of cognitive performance. The data is intended for exploratory analysis of relationships between daily habits and cognitive outcomes.

- **Notes / Ethical Considerations:**  
  - The dataset is publicly available and anonymized.  
  - All analyses in this project focus on **associations**, not causal claims.  
  - Results should be interpreted cautiously due to self-reported measures and potential confounding factors.
